## Overview

$10,000/month in free Google Search Ads under [[Common Good Network 501c3]].
Currently idle. Activating this creates a continuous inbound engine — new donors, KYCannaCon ticket buyers, directory traffic, and email list growth — all from people already searching for what KYNORML offers.

**Account:** Under Common Good Network (must use CGN's domain for landing pages)
**Status:** Idle — blocked until CGN website is live
**Grant type:** Google Ad Grants (search ads only, no display)
**Dependency:** CGN website is currently being built — GA4 installation and landing pages required before activation

---

## Before You Start — Account Health Check

Log into ads.google.com and verify:

- [ ] Account status is **Active** (not suspended)
- [ ] Billing is set to **Google Ad Grants** (not a paid method)
- [ ] No policy violations flagged

If the account shows a suspension warning, fix it before building campaigns — suspended accounts reject all new campaigns.

---

## Step 1 — Conversion Tracking (Required)

Google will suspend the grant without active conversion tracking.

**What counts as a conversion:**
- Donation button click (Zeffy link)
- KYCannaCon ticket purchase
- Email list signup
- Directory page visit (secondary)

**Setup path:**
1. Confirm CGN's website has **Google Analytics 4 (GA4)** installed
2. In Google Ads: Tools → Conversions → + New Conversion Action
3. Link GA4 to Google Ads account
4. Set "Donate" click and "Register" click as primary conversions
5. Set email signup as secondary conversion

**Blocker:** CGN website is still being built. Google Ads cannot go live until the site is up and GA4 is installed. This is the gating dependency for all of Track 3.

## Step 3 — Grant Requirements (Stay Compliant)

| Requirement | Target | Action if failing |
|---|---|---|
| Account CTR | ≥ 5% across all campaigns | Pause low-CTR keywords, tighten ad groups |
| Active campaigns | ≥ 1 active campaign at all times | Never let all campaigns pause |
| Conversion tracking | Must be active | Verify monthly |
| Landing page quality | Must be relevant + functional | Test pages after any site update |
| No single-word keywords | Except brand terms | Review keyword list quarterly |

---

## Step 4 — Monthly Review Checklist

- [ ] Check overall CTR — above 5%?
- [ ] Check conversion data — which campaigns are converting?
- [ ] Pause keywords with 0 clicks in 30 days
- [ ] Add negative keywords for irrelevant traffic
- [ ] Confirm all landing pages are working
- [ ] Review search terms report — any new keywords to add?
