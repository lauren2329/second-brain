## The Order of Operations — Why This Sequence

1. **Directory listings first** — the businesses are already there, the list is already built. This is the fastest cash available with the least setup. Do it today.
2. **Sponsor calls second** — you have warm dispensary contacts. A phone call this week can land $8,500 upfront. That is the single highest-value action available to you right now.
3. **Membership third** — needs one hour of Zeffy setup, then one email send. $25/month compounds.
4. **Chapter outreach fourth** — the VMP income engine. Starts paying in 30–45 days. Begin building the list now so it's ready when you are.

None of these require waiting. None require a website to be built. None require the CGN site. Start today.
